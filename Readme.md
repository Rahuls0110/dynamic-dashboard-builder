 Dynamic Excel Dashboard

A simple web-based dashboard that allows users to upload Excel or CSV files and generate interactive charts, KPI cards, and PDF reports dynamically.

Features:

Upload Excel (.xlsx, .xls) or CSV files
Generate:
Bar charts
Line charts
Pie charts
Summary cards
KPI comparison widgets

Aggregation options:
Sum
Average
Count
Download individual widget as PDF
Download all widgets as a single PDF
Responsive dashboard layout

Technologies Used

HTML
CSS
JavaScript (ES6)
Chart.js
SheetJS (XLSX)
jsPDF
html2canvas

How to Run
Download or clone the repository
Open index.html in your browser
Upload an Excel or CSV file
Select chart type, axes, and aggregation
Generate charts and download reports
No backend required. Runs completely in the browser.

This project demonstrates:
Dynamic data handling
Aggregation logic (sum/avg/count)
KPI trend calculation using date comparison
Interactive chart rendering

PDF generation from charts and dashboard widgets
